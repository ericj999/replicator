逐步快速入门

从智能手机或平板电脑备份照片到个人计算机

  1. 点击菜单上 执行项目→创建... 选项或工具栏的 + 打开创建执行项目对话框。
    或
  2. 点击第一个 … 选择便携设备上的照片文件夹。
  3. 点击下一个 … 选择在硬盘上目标文件夹。 你也可以点击 新增文件夹 在目前所选的文件夹创建新子文件夹来存储照片。
  4. 输入执行项目名称并按下 创建 建立该任务项目。
  5. 要执行项目时，点击菜单上 执行项目→执行 选项或工具栏的 ▸。
    或

将音乐从个人计算机传输到智能手机或平板电脑

  1. 点击菜单上 执行项目→创建... 选项或工具栏的 + 打开创建执行项目对话框。
    或
  2. 点击第一个 … 打开文件资源管理器并选择音乐文件夹。
  3. 点击下一个 … 打开文件资源管理器并选择选择便携设备上的音乐文件夹。
  4. [可选步骤] Windows 音乐文件夹可能包含某些无法复制到便携设备上的音乐文件夹的系统文件。 您可以使用高级选项配置文件案类型筛选器仅复制音乐文件。 点击 高级选项 打开"高级选项"对话框。 点击 包括以下文件类型 单选按钮并在编辑框中输入音乐媒体文件类型并点击 确认 设定该选项。
  5. 输入执行项目名称并按下 创建 建立该任务项目。
  6. 要执行项目时，点击菜单上 执行项目→执行 选项或工具栏的 ▸。
    或

注意: 音乐传输在苹果设备上无法使用。

高级选项

  • 选择来源文件的类型

    • 所有文件: 所有文件都将复制到目标文件夹中。
    • 包括以下文件类型: 只有与指定文件类型匹配的来源文件才会被复制。
    • 排除下列文件类型: 将复制除指定文件类型之外的所有文件。

  • 选择如何在目标文件夹中组织文件

    • 与选定的来源文件夹以下的结构相同: 目标文件夹下子档案夹和档案与选定来源文件夹具有相同的布局。 例如: 选定来源文件夹是 "This PC\iPad\Internal Storage\DCIM", 目标文件夹是 "C:\My Backups\iPad Photos"。 文件在 "This PC\iPad\Internal Storage\DCIM\100APPLE" 文件夹将会被复制到 "C:\My Backups\iPad Photos\100APPLE" 文件夹下。
    • 使用从来源文件夹根开始的相同结构: 原始文件將以完整的路徑被覆制到目的文件夾下。 例如: 选定来源文件夹是 "This PC\iPad\Internal Storage\DCIM", 目标文件夹是 "C:\My Backups\iPad Photos"。 文件在 "This PC\iPad\Internal Storage\DCIM\100APPLE" 文件夹将会被复制到 "C:\My Backups\iPad Photos\This PC\iPad\Internal Storage\DCIM\100APPLE" 文件夹下。
    • 使用以下格式按日期对文件分组: 此选项只适用于目标文件夹位于磁盘驱动器上。 档案将被覆制到目标文件夹中由定义的模板格式化的子档案夹中。 例如: 假设定义的模板为 "$(YYYY)\$(MMMM)" 和目标文件夹是 "C:\My Backups\iPad Photos", 2018年2月的照片将复制到 "C:\My Backups\iPad Photos\2018\February" 檔案夾下。

问题诊断

  • 无法将文件从便携设备复制到硬盘上: 请确保便携设备已连接到个人计算机, 并且启用了媒体传输模式(MTP)模式。
  • 无法从便携设备中选择文件夹: 请确保显示了便携设备, 并且可以从 Windows 文件资源管理器查看文件夹。

系统要求
