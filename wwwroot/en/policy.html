File Replicator Privacy Policy

AQITEK (“Company”, “we”, “us”) is the company that develops File Replicator. File Replicator is a Windows* desktop application and intended for use as is.

File Replicator (the “Software”) does not collect any personal information. The Software does not send any data over the Internet. However, the Software uses the “Documents” folder on the Software installed system to store the configuration data that are required for the Software to function properly. Log data are logged only for Software debugging purpose. No log data are collected by us.

Your continuous use of this constitutes your agreement to be bound by such changes to this privacy policy. You must stop using the Software if you do not accept the terms of this policy.

Any emails sent to us will not be disclosed or shared with any third parties. Any ideas and feature requests shared in the emails become the solo properties of the Company.

If you have questions regarding to this privacy policy, you may contact us by sending email to support@aqitek.com.
