Step-by-Step Quick Start

Backup photos from smartphone or tablet to PC

  1. Click on Task→New... on menu or + on the toolbar to open create dialog.
    or
  2. Click on first … to select photo files folder on your portable device.
  3. Click on next … to select the destination folder on a local drive. You can also right click on a selected folder to create a sub-folder in File Explorer for storing the photos.
  4. Enter task name and click on Create to create the task.
  5. To run a task, click on Task→Run on menu or ▸ on the toolbar to run the task.
    or

Transfer music from PC to phone or tablet

  1. Click on Task→New on menu or + on the toolbar to open create dialog.
    or
  2. Click on first … to open File Explorer and select Music folder.
  3. Click on next … to open the File Explorer and select the Music folder from your portable device.
  4. [Optional Step] The Windows Music folder may contain some system files that cannot be copied to the music folder on the portable device. You can specify file type filters to only copy the music files using the advanced options. Click on Advanced Options to open the Advanced Options dialog. Check the Include the following file type(s) radio button and enter the music media file types in the edit box and click on OK to apply the option.
  5. Enter task name and click on Create to create the task.
  6. Click on Task→Run on menu or ▸ on the toolbar to run the task.
    or

Note: Music transfer does not work on Apple devices.


More Topics

Advanced Options

  • Select types of source files

    • All files All the files will be replicated to the destination folder.
    • Include the following file types Only the source files matched the specified file types will be replicated.
    • Exclude the following file types All the files except the specified file types will be replicated.

  • Select how files are organized in the destination folder

    • Use same folder structure starting after the selected source folder The sub-folders under selected source folder will be created under the destination folder with same layouts. e.g. The source folder is "This PC\iPad\Internal Storage\DCIM", the destination folder is "C:\My Backups\iPad Photos". After the replication, files in "This PC\iPad\Internal Storage\DCIM\100APPLE" folder are copied to "C:\My Backups\iPad Photos\100APPLE" folder.
    • Use same folder structure starting from the root of the selected source folder The source files are copied to the destination folder with full path of source paths. e.g. The source folder is "This PC\iPad\Internal Storage\DCIM", the destination folder is "C:\My Backups\iPad Photos". After the replication, files in "This PC\iPad\Internal Storage\DCIM\100APPLE" folder are copied to "C:\My Backups\iPad Photos\This PC\iPad\Internal Storage\DCIM\100APPLE" folder.
    • Group files by date using the following format This option is only available when the destination folder is on a disk drive. The files will be copied to the sub-folder formatted by the defined template in destination folder. e.g. If the defined template is "$(YYYY)\$(MMMM)" and destination folder is "C:\My Backups\iPad Photos", a photo created in February 2018 will be copied to "C:\My Backups\iPad Photos\2018\February" folder.

Troubleshooting

  • Replication failed from portable device to local hard disk: Make sure the portable device is connected to PC and the media transfer mode (MTP) mode is enabled.
  • Cannot select folder from a portable device: Make sure the portable device is displayed and the folders can be viewed from Windows File Explorer.

System Requirements

  • Windows 10 Desktop version
