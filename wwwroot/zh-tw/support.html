逐步快速入門

從智慧手機或平板電腦備份照片到個人電腦

  1. 點擊功能表上 執行項目→新建... 選項或工具欄的 +打開新建執行項目對話方塊。
    或
  2. 點擊第一個 … 選擇可攜式裝置上的照片資料夾。
  3. 點擊下一個 … 選擇在硬碟上目的資料夾。 你也可以點擊 新增資料夾 在目前所選的資料夾建立新子資料夾來存儲照片。
  4. 輸入執行項目名稱並按下 建立 建立該任務項目。
  5. 要執行項目時，點擊功能表上 執行項目→執行 選項或工具欄的 ▸。
    或

將音樂從個人電腦傳輸到智慧手機或平板電腦

  1. 點擊功能表上 執行項目→新建... 選項或工具欄的 +打開新建執行項目對話方塊。
    或
  2. 點擊第一個 … 打開檔案總管並選擇音樂資料夾。
  3. 點擊下一個 … 打開檔案總管並選擇選擇可攜式裝置上的音樂資料夾。
  4. [可選步驟] Windows 音樂資料夾可能包含某些無法複製到可攜式裝置上的音樂資料夾的系統檔。 您可以使用高級選項設定檔案類型篩選器僅複製音樂檔。 點擊 進階選項 打開"進階選項"對話方塊。 點擊 包括以下檔案類型 選項按鈕並在編輯方塊中輸入音樂媒體檔案類型並點擊 確定 設定該選項。
  5. 輸入執行項目名稱並按下 建立 建立該任務項目。
  6. 要執行項目時，點擊功能表上 執行項目→執行 選項或工具欄的 ▸。
    或

注意: 音樂傳輸在蘋果設備上無法使用。

進階選項

  • 選擇原始檔案的類型

    • 所有檔案: 所有檔都將複製到目的資料夾中。
    • 包括以下檔案類型: 只有與指定檔案類型匹配的原始檔案才會被覆制。
    • 排除下列檔案類型: 將複製除指定檔案類型之外的所有檔案。

  • 選擇如何在目的資料夾中組織檔案

    • 與選定原始資料夾以下的結構相同: 目的資料夾下子資料夾和檔案與選定原始資料夾具有相同的佈局。 例如: 選定原始資料夾是 "This PC\iPad\Internal Storage\DCIM", 目的資料夾是 "C:\My Backups\iPad Photos"。 檔案在 "This PC\iPad\Internal Storage\DCIM\100APPLE" 資料夾將會被複製到 "C:\My Backups\iPad Photos\100APPLE" 資料夾下。
    • 使用從原始資料夾根目錄開始的相同結構: 原始檔案將以完整的路徑被覆制到目的資料夾下。 例如: 選定原始資料夾是 "This PC\iPad\Internal Storage\DCIM", 目的資料夾是 "C:\My Backups\iPad Photos"。 檔案在 "This PC\iPad\Internal Storage\DCIM\100APPLE" 資料夾將會被複製到 "C:\My Backups\iPad Photos\This PC\iPad\Internal Storage\DCIM\100APPLE" 資料夾下。
    • 使用以下格式按日期對檔案分組: 此選項只適用於目的資料夾位於磁碟機上。 檔案將被覆制到目的資料夾中由定義的範本格式化的子資料夾中。 例如: 假設定義的範本為 "$(YYYY)\$(MMMM)" 和目的資料夾是 "C:\My Backups\iPad Photos", 2018年2月的照片將複製到 "C:\My Backups\iPad Photos\2018\February" 資料夾下。

問題診斷

  • 無法將檔從可攜式裝置複製到硬碟上: 請確保可攜式裝置已連接到個人電腦, 並且啟用了媒體傳輸模式(MTP)模式。
  • 無法從可攜式裝置中選擇資料夾: 請確保顯示了可攜式裝置, 並且可以從 Windows 檔案總管查看資料夾。

系統要求

  • Windows 10 Desktop version
