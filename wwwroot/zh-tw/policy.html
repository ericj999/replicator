File Replicator 隱私權政策

AQITEK （“公司”，“我們”）是開發 File Replicator 的公司。 File Replicator 是一個 Windows* 桌面應用程序，旨在按原樣使用。

File Replicator （“軟體”）不會收集任何個人信息。本軟體不通過 Internet 發送任何數據。但是，本軟體使用軟體安裝系統上的“文檔”文件夾來存儲軟體正常運行所需的配置數據。日誌數據僅用於軟體調試目的。我們沒有收集任何日誌數據。

我們可能會定期更新此政策。您可以在 http://www.aqitek.com/zh-tw/policy.html上查看軟體私人政策的當前版本。您持續使用本協議即表示您同意接受本隱私政策的此類更改。如果您不接受本政策的條款，則必須停止使用本軟體。

發送給我們的任何電子郵件都不會透露或與任何第三方共享。電子郵件中共享的任何想法和功能請求都將成為公司的財產。

如果您對本隱私政策有任何疑問，可以發送電子郵件至 support@aqitek.com 與我們聯繫。
